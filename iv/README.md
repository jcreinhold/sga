## Editorial Conventions

- Original PDF page markers are preserved as bracketed markers such as `[PDF p. 311]`.
- Numbering of exposes, sections, definitions, propositions, exercises, formulas, and cross-references follows the
  source.
- Mathematical notation is written in TeX for KaTeX rendering. Inline mathematics uses `$...$`; displayed mathematics
  uses `$$...$$` or fenced `text` blocks only for diagrams whose faithful TeX reconstruction is uncertain.
- Historical terminology is preserved when modernization would change the mathematical or historical content.
- Translator notes are used only to prevent a concrete misunderstanding, especially where OCR damage or period
  terminology matters.
